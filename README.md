# geocode

Udemy Node.js course project using the `request` module.

## Description

This repository contains a simple Node.js project created as part of a Udemy course. It appears to focus on geocoding and HTTP requests using the `request` package.

## Getting started

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

## Running the project

Run the application with Node.js:

```bash
node app.js
```

> If the entry file is different, replace `app.js` with the correct filename.
